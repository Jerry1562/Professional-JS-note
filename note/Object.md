# JS的对象 #
本文主要记述JS对象操作的一些方法
## 对象属性 ##
对象的属性有两种类型：**数据属性**、**访问器属性**

|   数据属性   |  访问器属性  |
| :----------: | :----------: |
| configurable | configurable |
|  enumerable  |  enumerable  |
|   writable   |     set      |
|    value     |     get      |
